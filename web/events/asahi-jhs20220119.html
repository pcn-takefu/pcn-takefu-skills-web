便利なショートカットキー

Win + ←　　　：画面半分にウィンドウを広げる

Win + ↑　　　：画面全体にウィンドウを広げる

Fn + F2　　　：ファイルやフォルダの名前変更

Fn + F5　　　：ブラウザの再読み込み（リロード）

Ctrl + c　　　：選択されている文字をクリップボードにコピー

Ctrl + v　　　：クリップボードの内容を貼り付け

Ctrl + f　　　：検索

Ctrl + s　　　：上書き保存

Ctrl + z　　　：ひとつ戻る

問題文を変えてみる

1. VSCodeでquiz.htmlをひらき、「野村先生」で検索

2. 見つかった問題文の箇所の文章を書き換える

3. 上書き保存する

4. ブラウザ（Edge）をアクティブにし再読み込み（リロード）する

問題の画像を変えてみる

1. VSCodeでquiz.htmlをひらき、「クイズ画像」で検索

2. imgタグ内のsrcの中身を「./assets/img/question-image.png」に書き換える

3. 上書き保存する

4. ブラウザ（Edge）をアクティブにし再読み込み（リロード）する

5. 自分のPC内の画像を表示するには、ファイルを「[自分のフォルダ(101-01等)]/assets/img/」にコピーする

6. 手順2と同様にファイル名を変更する

7. 上書き保存する

8. ブラウザ（Edge）をアクティブにし再読み込み（リロード）する

選択肢を変えてみる

1. VSCodeでquiz.htmlをひらき、「クイズ選択肢」で検索

2. 選択肢のテキストを変更する（改行したい箇所には<br>タグをいれる）

3. 上書き保存する

4. ブラウザ（Edge）をアクティブにし再読み込み（リロード）する

正解を変えてみる

1. VSCodeでquiz.htmlをひらき、「クイズ選択肢」で検索

2. <input>タグ内のvalueを、正解ならvalue="1"、間違いの項目ならならvalue="0"にする

3. 上書き保存する

4. ブラウザ（Edge）をアクティブにし再読み込み（リロード）する

解説を変えてみる

1. VSCodeでquiz.htmlをひらき、「クイズ解説テキスト」で検索

2. 解説文のテキストを修正する

3. 上書き保存する

4. ブラウザ（Edge）をアクティブにし再読み込み（リロード）する

解説の画像を変えてみる

1. VSCodeでquiz.htmlをひらき、「クイズ解説画像」で検索

2. imgタグ内のsrcの中身をひとつ「./assets/img/question-image.png」に書き換える

3. 上書き保存する

4. ブラウザ（Edge）をアクティブにし再読み込み（リロード）する

5. 自分のPC内の画像を表示するには、ファイルを「[自分のフォルダ(101-01等)]/assets/img/」にコピーする

6. 手順2と同様に表示したい画像のファイル名に変更する

7. 上書き保存する

8. ブラウザ（Edge）をアクティブにし再読み込み（リロード）する

文字の色を変えてみる

1. VSCodeで assets/css/style.css をひらき、「クイズテキスト」で検索

2. 「.l-question-text」の括弧内「color: #000000;」の「000000」を変更する

3. 上書き保存する

4. ブラウザ（Edge）をアクティブにし再読み込み（リロード）する

ボタンの形を変えてみる

1. VSCodeで assets/css/style.css をひらき、「クイズ選択肢」で検索

2. 「.l-question-item>label」の括弧内「/* border-radius: 10px; */」の前後の「/*」「*/」を消して、「border-radius: 10px;」にする

3. 上書き保存する

4. ブラウザ（Edge）をアクティブにし再読み込み（リロード）する

タイトルを中央寄せにしてみる

1. VSCodeで assets/css/style.css をひらき、「h1.l-header-title」で検索

2. 「h1.l-header-title」の括弧内に「text-align: center;」を1行追加する

3. 上書き保存する

4. ブラウザ（Edge）をアクティブにし再読み込み（リロード）する

文字サイズを変えてみる

1. VSCodeで assets/css/style.css をひらき、「クイズ解説テキスト」で検索

2. 「.l-answer-text p」の括弧内「font-size: 2.5rem;」の数値を変更する

3. 上書き保存する

4. ブラウザ（Edge）をアクティブにし再読み込み（リロード）する

背景色を変えてみる

1. VSCodeで assets/css/style.css をひらき、「h1.l-header-title」で検索

2. 「h1.l-header-title」の括弧内「background-color: #FFFFFF;」の前後の「FFFFFF」の値を変更する(0〜9,A〜Fの16進数、先頭から2桁ずつ、赤緑青(RGB)の色になる

3. 上書き保存する

4. ブラウザ（Edge）をアクティブにし再読み込み（リロード）する
